# Contributing to DevFreebies

Thank you for your interest in contributing to DevFreebies.  
This project is built for the developer community, by the developer community.

## How you can contribute

You can help in many ways:

- Submitting new free tools and resources
- Fixing bugs
- Improving UI/UX
- Adding new features
- Improving documentation

## Getting started

1. Fork the repository
2. Clone your fork
3. Create a new branch  
   git checkout -b feature/your-feature-name
4. Make your changes
5. Commit with a clear message  
   git commit -m "Add new feature"
6. Push to your fork
7. Open a Pull Request

## Code style

- Use clean, readable code
- Follow existing formatting and structure
- Keep UI components consistent with the current design
- Avoid breaking existing functionality

## Submitting resources

If you are adding a new tool or resource:
- It must be free or have a useful free tier
- It must be relevant to developers
- Avoid spam or promotional-only content

## Review process

All pull requests are reviewed before merging.  
We may request changes to ensure quality and consistency.

Thank you for helping make DevFreebies better.
