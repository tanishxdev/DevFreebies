# Code of Conduct

DevFreebies is an open-source project built to support and grow the developer community.

We expect everyone participating in this project to follow these rules:

## Be respectful

- Treat everyone with respect
- No harassment, hate speech, or discrimination
- Be kind, even when you disagree

## Be constructive

- Give helpful, clear feedback
- Focus on improving the project
- Avoid personal attacks

## No spam or abuse

- Do not submit spammy tools
- Do not abuse the platform or its users

## Responsibility

Maintainers have the right to remove any content, user, or contribution that violates these rules.

By contributing to DevFreebies, you agree to follow this code of conduct.
